== Description ==

This is a korean board. Simple and basic.

== Installation ==

e.g.

1. Upload `amumu-board` to the `/wp-content/plugins/` directory
2. Create folder `/wp-content/plugins/amumu-board/uploads/`
3. Activate the plugin through the 'Amumu-Board' menu in WordPress
4. Create page and select the page in 'Amumu-Board' menu.
